<h2>DFS vs BFS</h2>
<p>深さ優先探索（DFS）は進めるうちは進み、行き止まりに来たら分岐点まで戻る<br>訪問順序を取ったり、状況の遷移に対応するのが得意</p>
<br>一方で、幅優先探索はスタートから深さを揃えて探索するため、全ノードの最短距離がわかる
